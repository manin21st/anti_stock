새로 감시종목 탭을 추가하고 싶어. 위치는 포트폴리오 우측에 추가해줘
기본적으로는 api 를 통해서 불러오고 추가, 삭제하는 기능이 제공되면 좋겠어

추가될 감시종목 탭 화면 구성은 좌우 패널 형태로 3:7 비율로 나누면 돼
버튼은 좌측 패널 우측상단에 새로고침 버튼이 필요하고 우측 패널 우측상단에 추가, 삭제, 저장 버튼이 필요해

좌측 패널 좌측상단에는 종목을 검색하는 입력란이 필요해
종목명 입력란 바로 아래는 종목리스트를 코스피+코스닥 시장 종목들로 자동 나열해주면 돼
종목명 입력란에 한 글자씩 입력하면 하단 종목리스트가 실시간으로 필터링 되면 좋겠고
리스트에는 간단하게 종목코드, 종목명이 표시되면 되고 단일 및 다중 선택이 가능해야해

우측 패널에는 이미 저장되었거나 좌측 패널에서 선택되어 넘어 온 감시종목 리스트를 표시하는데
리스트에 표시할 정보는 NO, 종목명, 종목코드, 현재가, 등락률, MA20가격, 스파크라인 을 표시하면 돼
스파크라인은 구글스프레드시트 기능 같은건데 최근 한달간의 일봉 데이터를 표시하면 좋겠어

감시종목 리스트에 추가하려면 좌측 패널 종목리스트에서 선택된 상태로 우측 패널 추가 버튼을 눌러야돼

감시종목 탭이 추가되면 환경설정 탭 수동 종목 지정도 필요없으니 사라져야겠지
화면 디자인 톤앤매너는 기존 방식을 따르도록 해줘
